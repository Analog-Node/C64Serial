# C64 User Port Pin Mapping

This document defines how C64 user port signals are mapped to RS-232 signals.

## Target Signals

| C64 User Port | Function | RS-232 |
|--------------|----------|--------|
| PA2          | TXD      | TX     |
| PB0          | RXD      | RX     |
| GND          | Ground   | GND    |
